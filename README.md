## 🚀 Getting Started

### Prerequisites

- Python 3.9+
- `pip` or `uv` for package management
- API keys for relevant AI providers (see individual app READMEs)

### Running an App

Each application lives in its own directory with a dedicated `README.md`. Navigate to the app folder and follow its setup instructions:

```bash
# Clone the repository
git clone https://github.com/awesome-ai-apps/awesome-ai-apps.git
cd awesome-ai-apps

# Navigate to a specific app
cd <app-directory>

# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py
```

## 🤝 Contributing

We welcome contributions! Please read our [contribution guidelines](.github/PULL_REQUEST_TEMPLATE.md) before submitting a PR.

### Adding a New App

1. Fork this repository
2. Create a new directory for your app following the naming convention: `category/your-app-name/`
3. Add your app code along with a `README.md` using the [README template](.github/README_TEMPLATE.md)
4. Submit a Pull Request

### Reporting Issues

Found a bug or have a feature request? Please [open an issue](https://github.com/awesome-ai-apps/awesome-ai-apps/issues/new/choose) using one of our templates.
